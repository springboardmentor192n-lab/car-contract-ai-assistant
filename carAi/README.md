# Car Lease AI - Frontend

This is the Flutter frontend for the Car Lease AI Assistant.

For full project documentation, including setup and backend instructions, please see the [root README](../../README.md).

## Quick Start (Frontend Only)

1.  **Install Dependencies**:
    ```bash
    flutter pub get
    ```

2.  **Run with API Connection**:
    Ensure the backend is running on `localhost:8000` (or `10.0.2.2:8000` for Android Emulator).
    ```bash
    flutter run
    ```
